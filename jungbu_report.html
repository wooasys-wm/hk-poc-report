3. 시나리오 (운영안 정의)

본 검토에서는 세 가지 cost 정책 시나리오를 정의한다.

3.1 시나리오 개요

시나리오 의도 비고
실단가 기준 배송 단가표 가준으로 최적 사용 유도 운임비가 톤급간 비용 차이를 크게 벌리지 않음. 예들들어 91~100km 구간 1T > 3.5T 전환시 약 1.3배 이지만 적재 가능량은 훨씬 커짐. 1T를 사용하지 않는 이유로 작용.
3.5T/5T 유도 3.5T / 5T 우선 사용 유도 3.5T / 5T 적극 사용할 수 있도록 유도하는 시나리오.
1T/3.5T 유도 1T / 3.5T 우선 사용 유도 1T / 3.55T 적극 사용할 수 있도록 유도하는 시나리오.

3.2 시나리오별 기대 효과

  • 실단가 기준 — 실 단가표에 가장 가까운 cost. 8T / 11T 통합으로 routes 수 최소화.
  • 3.5T/5T 유도 — 3.5T / 5T 풀가동, 1T / 11T 사용 최소화. 중형 차 위주 운영안.
  • 1T/3.5T 유도 — 1T 풀가동, 분산 운영. routes 수는 가장 많지만 실 운임 패턴 (1T 중심 As-Is) 에 가장 가까움.

4.3 해석

  • As-Is 실 운임은 1T 중심 구조라 1T 총액 비중이 크다.
  • 실단가 기준 은 큰 차 통합이 강해 8T / 11T 비중이 커지고 총 운임이 가장 낮다.
  • 3.5T/5T 유도 는 3.5T / 5T 비용 비중이 커진다.
  • 1T/3.5T 유도 는 1T 분산 운영으로 1T 비용 비중이 다시 커지고, 총 운임비가 실 운임에 가장 가깝다.

5. 검증 대상 차량 구성

톤급 순서는 모두 1T / 3.5T / 5T / 8T / 11T.

  • 상시 차량 구성안 (B) — 평상시 운영용. 27일 중 대부분 일자에 적용되는 기본 차량 구성. 피크 일자 (배송량 집중일)에 미배차건 발생

8. 결론

8.1 상시 차량 구성 + Peak 차량 구성 2단계 구조

미배차 0 을 만드는 단일 고정 차량 구성을 상시 차량 구성으로 보는 것은 과도하다. 평상시를 담당하는 상시 차량 구성 + 피크 일자를 흡수하는 Peak 차량 구성의 2단계 구조가 자연스럽다.

8.2 8T/11T 고정 제약 하, 중소형 차량 증차를 통한 미배차 해소

1차 Peak 차량 구성 검증에서는 세 가지 변형안 모두 3.5T·5T·8T·11T 차량이 동시 만재 상태에 도달하며 미배차 발생함.

반면, 8T(9대) 및 11T(4대)의 상위 톤급 차량 제약을 유지한 상태에서 3.5T·5T 차량(필요 시 1T)을 추가 투입한 결과, 모든 변형안에서 미배차 0건 달성이 가능.

필요 증차 규모는 1차 검증 대비 +3~+5대 수준으로 확인됨.
